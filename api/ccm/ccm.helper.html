<static> integrate( [priodata] [, dataset])

integrate priority data into a dataset

Each value of each property in the given priority data will be set in the given dataset for the property of the same name. A priority data property with the explicit value undefined provokes to delete the same property in the given dataset. With no given priority data, the result is the given dataset. With no given dataset, the result is the given priority data.

Parameters:
Name Type Argument Description
priodata object <optional>

priority data
dataset object <optional>

dataset
Returns:

dataset with integrated priority data

Type
object
Examples
var dataset  = { firstname: 'John', lastname: 'Doe', fullname: 'John Doe' };
var priodata = { lastname: 'Done', fullname: undefined };
var result = ccm.helper.integrate( priodata, dataset );
console.log( result );  // { firstname: 'John', lastname: 'Done' };
var result = ccm.helper.integrate( { value: 'foo' } );
console.log( result );  // { value: 'foo' }
var result = ccm.helper.integrate( undefined, { value: 'foo' } );
console.log( result );  // { value: 'foo' }

<static> val(string [, regex])

validate a given string

This function returns the given string only if it matches the given regular expression. The result is null if the given string not matches the given regular expression. The result is the return value of ccm.helper.htmlEncode(string) if the given regular expression equals true (string === true). The result is the return value of ccm.helper.noScript(string) if the optional given regular expression is a falsy value (false, null, undefined, 0, NaN, '' or ""). The given regular expression passes ccm.helper.regex(regex) if the given regular expression is a string.

Parameters:
Name Type Argument Description
string string

given string
regex ccm.types.regex | string | boolean <optional>

given regular expression
Returns:

returns given string or null

Type
string
Examples
ccm.helper.val( 'Hello, world!', /^[A-Z][a-z ,!]*$/ );  // => 'Hello, world!'
ccm.helper.val( 'Hello, world!', /^[A-Z][a-z]*$/ );     // => null
ccm.helper.val( 'Hello, world!<script>alert("XSS");</script>', true );
// => 'Hello, world!<script>alert("XSS");</script>' (no alert call)
ccm.helper.val( 'Hello, world!<script>alert("XSS");</script>' );
// => 'Hello, world!' (no alert call)
